# Convertidor Celsius → Fahrenheit

Archivos creados en `conversor-temperatura`:

- `index.html` — interfaz principal
- `styles.css` — estilos y diseño responsivo
- `script.js` — lógica de conversión

## Cómo usar

1. Abre la carpeta `conversor-temperatura`.
2. Abre `index.html` en tu navegador (doble clic o desde PowerShell):

```powershell
cd .\conversor-temperatura
Start-Process index.html
```

3. Ingresa el valor en Celsius y presiona "Convertir" o Enter.
